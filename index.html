Cómo usar el Simulador

1. Selecciona los vehículos: Elige el tipo de vehículo para el lado A (izquierda) y B (derecha). Cada uno tiene diferente masa.

2. Define las velocidades: Ingresa la velocidad en km/h. El Vehículo A viaja hacia la derecha y el B hacia la izquierda.

3. Coeficiente de Restitución: Define qué tan "elástico" es el choque (ver ayuda específica).

4. Simular: Presiona el botón para ver la colisión y los resultados físicos.

Coeficiente de Restitución (e)

Es un número entre 0 y 1 que indica qué tanta energía cinética se conserva.

  • 1 (Elástico): Los objetos rebotan perfectamente (como bolas de billar).
  • 0 (Inelástico): Los objetos se quedan pegados tras el choque (como plastilina).
  • 0.5 (Realista): Los autos se deforman y pierden energía.
